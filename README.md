# Evaluating Student Performance Factors Using Linear Regression
This project showcases data cleaning and analysis techniques to explore factors that could affect student performance which is indicated by students' final exam scores. This project analyzes a dataset sourced from Kaggle as part of a learning exercise.  
The data is used to demonstrate analytical techniques and modeling rather than to produce real-world insights.
